🤖 AI Career Counsellor
🚀 Overview

Built an AI-powered career guidance platform to help students and professionals make data-driven career decisions through personalized recommendations and structured roadmaps.

🎯 Problem
Users lack clarity on:

Which career to choose
What skills are required
How to transition into new roles
👥 Target Users
Students: Need direction
Early professionals: Need clarity
Career switchers: Need structured transition plans

💡 Solution
Developed an AI system that:

Maps user interests & skills → relevant careers
Identifies skill gaps
Generates personalized learning roadmaps

⚙️ How It Works
User inputs profile (skills, interests, goals)
AI evaluates using scoring model

Outputs:
Top career matches
Skill gap analysis
Step-by-step roadmap

🧠 Decision Logic
Career Match Score:
(Interest 30%) + (Skill 25%) + (Market Demand 20%) + (Salary 15%) + (Ease of Learning 10%)

Skill Gap:
Required Skills − Current Skills

📊 Impact (Expected / Sample)
⬆️ Improved career decision clarity
⬆️ Reduced research time for users
⬆️ Higher confidence in career paths
